Overview:

Full-stack Client Lead Management System

Simulates a real-world CRM used by businesses

Designed to capture, manage, and track leads efficiently

Built as a practical, real-world business application

Features:

Secure admin login and logout

Add new leads with validation

Update lead status (New, Contacted, Converted)

Add and save notes per lead

Delete leads

Search leads by name, email, or status

Sort leads by date, status, or source

Filter leads by status

Analytics dashboard (Total, Contacted, Converted)

Color-coded lead cards by status

Responsive design for mobile devices

Error handling and API validation

Tech Stack:

HTML

CSS

JavaScript (Vanilla JS)

Node.js

Express.js

MySQL

Key Learnings:

CRUD operations (Create, Read, Update, Delete)

RESTful API development

Database integration with MySQL

Authentication handling

Frontend–backend communication

Form validation and error handling

Building a functional data-driven business system

Future Improvements:

Multiple admin accounts

Password hashing (bcrypt)

Lead archiving instead of deletion

Export leads to CSV

Email notifications

Enhanced UI animations and design improvements
